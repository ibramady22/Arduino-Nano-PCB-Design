# Arduino Nano PCB Design

This repository contains a **custom Arduino Nano PCB design** created using **Altium Designer**.  
It includes the complete schematic and PCB layout for an Arduino Nano–compatible board, suitable for learning, modification, or fabrication.

## Features
- Arduino Nano–compatible pinout
- Complete schematic and PCB layout
- Altium Designer project files
- Easy to customize for other projects

## Repository Contents
- Schematic files (`.SchDoc`)
- PCB layout files (`.PcbDoc`)
- Altium project file (`.PrjPcb`)
- Custom libraries (symbols & footprints)
